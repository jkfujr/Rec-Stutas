## 使用说明

1. 需要Python 3.10及以上版本

2. 安装依赖

```
pip install -r requirements.txt

# 支持SSL (可选)
pip install SSLContext
```

3. 添加 API

```
# 编辑 `./config/rec_api.py` 文件


```

4. 运行

```
python main.py
```

5. 访问 [http://127.0.0.1:1111](http://127.0.0.1:11111) (默认)
